## 任务目标
- 实现登陆页面功能，实现常规的注册和登录功能
### 登陆页面
 - 必须输入：手机号或者邮箱
 - 必须输入：pwd
 - 登录按钮，要交验是否输入，登陆返回成功后，进入首页
### 注册功能
 - 填写 手机号，用户名称,邮箱，密码【需要二次确认】
 - 注册按钮，注册成功后，进入首页
